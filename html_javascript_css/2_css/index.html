Dette er HTML med CSS

HTML er indhold, CSS er form

CSS kontrollerer layout og typografi. Desuden tager det hånd om..

  • forskellige skærmstørrelser og devices - responsivt layout
  • at adskille udseende og indhold. Og det er smart hvis man engang beslutter sig for at man vil lave indholdet om.

CSS er så effektivt på grund af HTML dokumenters struktur. Ved hjælp af elementer, klasser og id'er kan man få fat i elementer i HTML helt ned til hver enkelt sætning.

Avanceret CSS

Når man bliver lidt skrappere kan man styre animationer og transitioner, få ting til at fade ind og ud eller forsvinde. Men først og fremmest kan man styre layout i store dokumenter uden at gå på kompromis med detaljen.

While CSS is great for creating text styles, it is helpful for formatting other aspects of Web page layout as well. For example, CSS can be used to define the cell padding of table cells, the style, thickness, and color of a table's border, and the padding around images or other objects. CSS gives Web developers more exact control over how Web pages will look than HTML does.
