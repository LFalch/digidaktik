Typografi

typografi, indtil ca. 1900 synonym for bogtryk eller bogtrykkerkunst i bredeste betydning. I nutiden betegnelse alene for satsfremstilling, herunder den grafiske formgivning (tilrettelæggelsen) af trykarbejdet, indskrivning af teksten (sætningen) samt ombrydning af satsen til færdige sider med placering af evt. illustrationer; denne sidste arbejdsgang hed montage i tidsrummet forud for de elektronisk styrede trykmetoder. Hele satsfremstillingsprocessen kaldes nu ofte prepress.

Den Store Danske, citeret fra Gyldendal, tilgået d. 11/10 2017

Som det fremgår af citatet, er betydningen af begrebet typografi blevet indsnævret noget gennem tiden. I denne tutorial arbejder vi derfor med det vi kunne kalde HTML sidens pre-press. Det vi ser på siden før designeren går i gang, er det materiale designeren har at arbejde med.
